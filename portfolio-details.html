SocialBD.NeT - Advanced Social Platform

demo.fortibd.com

SocialBD.NeT is a modern social networking platform designed to provide a seamless communication experience with integrated AI capabilities and real-time media sharing.

Throughout January 2026, I have been actively developing SocialBD.NeT, focusing on a scalable architecture that supports high-quality user interaction. The platform is built using PHP to ensure robust backend performance.

The primary challenge was implementing stable real-time communication tools and integrating an intelligent AI agent system that could operate beyond standard command sets.

I utilized the Groq API to build intelligent AI agents and implemented LiveKit to power reliable video and audio calls. This combination provides a modern, feature-rich social environment.

Key Features

  • Real-time Video/Audio Calls
  • AI-Powered User Agents
  • Secure Authentication
  • Dynamic User Stories
  • Interactive Newsfeed
  • Responsive Multi-device Support
